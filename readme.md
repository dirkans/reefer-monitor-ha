# Reefer Monitor Pro - Home Assistant
Integración oficial para leer datos de cámaras de frío Reefer Monitor Pro en Home Assistant.

### Instalación vía HACS
1. Abre HACS en tu Home Assistant.
2. Ve a Integraciones -> 3 puntitos -> Repositorios Personalizados.
3. Pega la URL de este repositorio y selecciona la categoría "Integración".
4. Descarga, reinicia Home Assistant, y añade la integración desde Ajustes -> Dispositivos y Servicios.
